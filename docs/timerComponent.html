    Méthode format

    Cette méthode prend un interval et le converti en une chaîne lisible.

      function format(total_ms) {
    const total_s = Math.floor(total_ms / 1000);
    const minutes = Math.floor(total_s / 60);
    let seconds = total_s - (minutes * 60);
    if (seconds < 10) {
      seconds = '0' + seconds;
    }
    return `${minutes}:${seconds}`;
  }

    La méthode onEnabled est appelée quand l’objet passe de l’état activé à désactivé.

        onEnabled() {
