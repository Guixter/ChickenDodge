    Méthode run

    Cette méthode initialise les différents systèmes nécessaires et démarre l’exécution complète du jeu.

      function run(config, launchScene) {
    module.config = config;
    return setupSystem(config)
      .then(() => {
        return launchGame(launchScene);
      });
  }

    Méthode launchGame

    Cette méthode initialise la scène du jeu et lance la boucle de jeu.

      function launchGame(launchScene) {
    return Utils.loadJSON(launchScene)
      .then((sceneDescription) => {
        return Scene.create(sceneDescription);
      })
      .then((scene) => {
        const displayFn = scene.display.bind(scene);
        const updateFn = scene.update.bind(scene);

        return Utils.loop([
          updateFn,
          displayFn,
        ]);
      });
  }

    Méthode globale requestFullScreen

    Cette méthode appelle la méthode correspondante du module graphicsAPI. Elle est appelée par le bouton de la page HTML. On l’assigne à l’objet document, ce qui permet d’exister dans le contexte global et donc d’être accessible depuis la page Web.

      document.requestFullScreen = function() {
    GraphicsAPI.requestFullScreen();
  };
